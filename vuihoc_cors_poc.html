PoC: CORS Exploit trên vuihoc.vn (Origin Reflection + Credentials)

Trang này giả lập trang web độc hại của hacker (ví dụ vuihoc-tang-hoc.com)

Hướng dẫn: Để PoC chạy được, bạn cần mở trang này từ domain thật (không phải file://).
Cách đơn giản: chạy python3 -m http.server 8000 trong folder chứa file HTML, rồi mở http://localhost:8000.
Lưu ý: Origin null (khi mở bằng file://) có thể bị chặn bởi một số browser.
